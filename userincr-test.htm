userincr jQuery plugin demo

userincr enhances edit boxes with "increment" and "decrement" buttons. The increment step can be chosen by the page user.

Download the latest version of the plugin from here

Default use: increment is +1 and generated title attribute.Change increment to 5 by entering +5↵ into the box.

Range is limited to 0—1000. Entering negative number changes the increment, rather than the value. Try -50

Range, increment and increment type is specified as DOM data-* attributes. Can be changed.

Custom button labels

Responsive to keyboard (up and down arrows) and mouse wheel; defined constants 'earth' and 'moon'.
